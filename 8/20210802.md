```
$(".xxx").메서드
```

1. `$ 함수`를 호출.
2. JQuery 객체의 `메모리 위치 주소값` 리턴. 
3. JQuery 객체는 `class=xxx` 를 가진 태그를 관리하고 있다. 

- `class=xxx` 는 0개 이상 일 수 있다. 없어도 JQuery 객체 생성되고, 여러개 일 수 있다. 


```
var 변수 = $(".xxx");

변수.메서드()
```
> 변수 안에 담고 사용 할 수 있다.  
> 반복해서 사용할 경우 변수에 넣어서 사용.
