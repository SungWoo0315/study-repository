```javascript
  // ----------------------------
  // num 에서 맨 뒤에서 부터 하나씩 낚아채서 result 변수 누적 시킴.
  // 이때, 3의 배수만큼 낚아챌 때 콤마(,)를 앞에 넣어 누적시킴
  // ----------------------------
  var cnt = 0;
  for ( var i=num.length-1; i>=0; i--) {
      cnt++;

      if( cnt%3==0 ){
          result = "," + num.charAt(i) + result;

      }else{
          result = num.charAt(i) + result;

      }
```
> 왼쪽으로 누적시켜야 하므로, `num.charAt(i)` 을 왼쪽에 더한다.   

> 맨앞에 붙는 `,` 도 제거해야 한다.  

> `indexOf()` 와 `substring()` 활용해서 적용.

#### # 앞에 0이 여러개 나올 경우를 걸러내려면? 0원 표시 할때 이외엔 0이 중첩되서 앞에 나오면 안된다.

```javascript
  if( num.length>0 ){
      num = parseInt(num,10) + "";    
  }
```
> 숫자문자를 숫자 정수로 바꿔준 후에 길이가 없는 문자데이터를 붙여준다.     
> 숫자문자에서는 0이 중첩되서 붙을수 있지만,    
> 정수는 0을 표현하는 0한개 빼고는 앞에서 000 이런식으로 중첩할수가 없다.  

> 길이가 없는 문자 `""` 를 붙여주어야 숫자문자로 다시 변환된다.       
> 숫자문자로 다시 변환 되어야, String 객체를 적용 할 수 있다.    

> 최소 숫자문자가 1개 이상 있어야 하는데, 숫자문자가 아무것도 없으면(공백만 들어가면) 에러가 난다. 공백울 숫자로 만들수 없기 때문, 프로그램 중단을 시켜야 한다.    

> 다른방식으로는 반복문으로 돌려서 `0` 과 `,` 를 제거해주면 된다.
