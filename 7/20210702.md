### <번호생략> 파일 경로 지정 방법의 종류에는 무엇이 있나요? 즉, 파일호출자 입장에서 피호출파일을 호출할 때 피호출파일의 경로를 어떻게 설정해야하나?   ★★★★ (입.)

경로없이 파일명만 있을때 => 같은 폴더 내에 있음.  

다른 폴더의 파일을 호출하고싶을때?   

**[상대경로]**, **[절대경로]** 사용은 상황에 따라 다르다.  

주로 [상대경로]를 쓰지만 절대적인것은 아니며, 상황에 따라 다르다.  

- **상대경로**  
  ```HTML  
  만약 C:a/b/xxx.html 파일안에서 아래 파일들을 호출할 경우 호출경로는?
  *******************************************************************
  피 호출파일 경로   => 호출파일 안에서 피 호출 파일을 호출할 때
  *******************************************************************
  C:a/b/YYY.html     => "YYY.html"
  -------------------------------------------------------------------
  C:a/zzz.html       => "../zzz.html"
  -------------------------------------------------------------------
  C:a/d/mmm.html     => "../d/mmm.html"
  -------------------------------------------------------------------
  C:a/b/c/nnn.html   => "c/nnn.html"
  -------------------------------------------------------------------

  호출하는 파일 기준으로 어디 들어있냐를 판단.

  <a href="boardRegForm.html">[새 글쓰기]</a><br><br>
  <a href="../boardRegForm.html">[새 글쓰기]</a><br><br>
  <a href="../b/boardRegForm.html">[새 글쓰기]</a><br><br>
  ```

```
Tip
웹서버에서    
맨앞에 `/` 로 시작할때는 동일한 웹서버를 기준으로 찾음.    
````

```
  ▶서버 쪽 언어(JSP, Java, PHP, ASP, .NET)을 사용하여 UI로부터 또는 UI로의 데이터의 이동, DB 연동 작업을 한다.
  ▶입력양식 관련 HTML 은 개발자가 할 수 도 있다.
  ▶복잡하고 어려운 Javascript 작업을 하는 경우도 있다.
```

### <13> a 태그가 뭐 할 때 사용 하나요?

```HTML  
------------------------------------------------------------------------------
●[문자열]이나 [이미지 태그]을 감싸고 [클릭]등과 같은 이벤트를 발생하면 다음의 기능을 구현하는 태그이다.
  [페이지 이동]
  [같은 문서내 특정위치 이동]
  [파일 다운로드]
  [이미지 출력]
  [음악.동영상 실행]
  [자바스크립트 코드 실행]
  [MS 아웃룩 실행] 등
------------------------------------------------------------------------------
●<참고>기본적으로 문자열에 마우스를 대면 자동으로 손모양이 보인다.
------------------------------------------------------------------------------
●<참고>이벤트
	-웹상에서 이벤트란 웹브라우저에서 발생하는 모든 움직임을 말한다.
	-주로 마우스나 키보드에의 발생하지만 다른 것에 의해서도 발생한다.
		예>html 문서가 웹브라우저에 로딩되는 것. html 문서가 웹브라우저에 언로딩되는 것(즉 사라지는 것)
	-이벤트의 종류
		load,unload, click, dbclick, mouseover, mouseout, mousedown, mousedragged, keydown
		keyup, keypress, focus, blur ,change 등등
------------------------------------------------------------------------------
------------------------------------------------------------------------------
<a href="서버자원의URL주소">문자열</a>
------------------------------------------------------------------------------
  ▶서버자원의URL주소 호출로 새로운 페이지 이동. 즉, 새 화면 열기
------------------------------------------------------------------------------
<a href="javascript:자바스크립트코드">문자열</a>
------------------------------------------------------------------------------
  ▶클릭 시 코딩한 [자바스크립트코드]가 실행된다.
  ▶보통 [자바스크립트코드]는 함수 호출 코딩이다.
------------------------------------------------------------------------------
<a href="다운로드파일명">문자열</a>
------------------------------------------------------------------------------
  ▶[파일 다운로드 대화상자]가 나타나고 확인 버튼 누르면 파일이 다운로드 된다.
  ▶<주의>웹에서 열어볼 수 있는 파일은 [파일 다운로드 대화상자]가
    뜨지 않고 새 화면을 열고 실행한다. 예>그림파일, txt 파일 등.
------------------------------------------------------------------------------
<a href="그림파일명">문자열</a>
------------------------------------------------------------------------------
  ▶이미지만 있는 화면으로 이동
------------------------------------------------------------------------------

------------------------------------------------------------------------------
  ▶방문자에 설치된 프로그램으로 음악 동영상 파일 재생.
```
